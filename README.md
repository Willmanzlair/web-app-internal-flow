# web-app-internal-flow

A practical study repo for understanding how modern web applications flow from frontend actions to backend decisions, API responses, sessions, tokens, and security testing observations.

This repo is written as a learning map. Each file follows the natural flow of a modern web application, from the first page a user sees, through registration, login, session restoration, and authenticated application behaviour.
